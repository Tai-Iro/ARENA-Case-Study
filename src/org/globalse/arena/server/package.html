This package contains all the classes realizing the arena server, in particular, providing remote objects for the interfaces in {@link org.globalse.arena.remote}.

The arena server is start with the StartArena class. The port for accepting requests, the class names of the games and tournament styles to be loaded, and (optionally) a code base from where to load the game-specific classes are specified in a properties file. The location of the properties file (URL or file name) is specified as the first argument on the command line.
