This package contains all the classes and interfaces needed to access the arena server remotely. This includes:

  • interfaces for remote objects:
    • RemoteArena
    • RemoteLeague
    • RemoteTournament
    • RemoteRound
    • RemoteMatch
    • RemoteLogger
  • remote listeners:
    • RemoteArenaListener
    • RemoteMatchListener
  • wrappers for transporting state:
    • LeagueInfo
    • TournamentInfo
    • MatchInfo
    • GameInfo

For the implementation of the remote interfaces and the main class starting the arena server, see the {@link org.globalse.arena.server} package.
